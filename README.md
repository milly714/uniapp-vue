# wechat_shop

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev:mp-weixin
npm run dev:h5
```

### Compiles and minifies for production
```
npm run build:mp-weixin
npm run build:h5
```

### Lints and fixes files
```
npm run lint
```
